# ⚖️ 申論實務雷達 exam-radar

為**調查局法律實務組**與**司法官・律師第二試**（皆申論題型）打造的每日情報站：
自動抓取「最新實務見解（大法庭／最高法院／憲法法庭）＋新法規／修法動態」，
也就是申論題最愛入題的命題熱區。

線上：https://elainechi-art.github.io/exam-radar/

## 涵蓋
- 🏛️ **憲法法庭最新判決**（直接爬 cons.judicial.gov.tw，含案由與全文連結）
- 📋 **新法規・修法動態**（法務部官方 RSS：全國法規最新異動＋修法新聞）
- 六科各三欄（最高法院／大法庭見解・修法・考點熱區）：
  刑法、刑事訴訟法、行政法、憲法、民法、商事法
- 📚 申論準備指南＋具參考價值的免費官方來源清單

## 考科對照
- 調查局法律實務組（三等）專業科目：刑法、刑事訴訟法、行政法、商事法（申論）
- 司法官・律師第二試：公法（憲法＋行政法）、民法、刑法、商事法（申論）

## 來源（全部免金鑰）
- Google News RSS（各科實務見解／修法／熱點查詢）
- 法務部 RSS：全國法規最新消息、新聞（台灣政府憑證需略過 SSL 驗證）
- 憲法法庭判決列表（靜態 HTML 解析）

## 運行
```bash
pip install -r requirements.txt
python radar_run.py    # 產出 docs/index.html
```
GitHub Actions `daily.yml` 每天 08:30（台灣）自動更新並 commit。
